* Ruby version
  - 2.3.1

* Database creation
  - `bin/rails db:create`
  - `bin/rails db:migrate`

* Database initialization
  - `bin/rails db:seed`

* How to run the test suite
  - `RAILS_ENV=test bin/rspec spec`

* About frontend
  - `cd frontend`
  - `npm install`
  - Use command `npm run watch` when developing.
